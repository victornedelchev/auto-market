# AutoMarket AI

AutoMarket AI is a multi-page car marketplace where users can publish, edit, and browse car listings. 
This project is built for a SoftUni capstone project.

## Technologies Used
* **Frontend**: HTML, CSS, Bootstrap 5, Vanilla JavaScript, Vite
* **Backend**: Supabase (Auth, Database, Storage)

## Project Structure
* `src/pages/` - Multi-page structure or logical page components
* `src/components/` - Reusable UI components
* `src/services/` - API and Supabase communication
* `src/utils/` - Helper functions
* `src/styles/` - Global styles and custom CSS
* `src/assets/` - Static files, images, etc.

## Setup Instructions
1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Build for production: `npm run build`
